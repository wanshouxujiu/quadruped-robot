一、安装注意事项
1、事先准备好螺丝刀和相应规格的内六角扳手
2、安装电路板时力量要适中
3、电路板USB口对自己，舵机的安装顺序为右边第一个顺时针，PA0~PA4为定时器2，控制四个高度舵机，PA6、PA7、PB0、PB1为定时器3，
控制4个转向舵机
4、安装舵机摇臂时要先全部初始化为90°，再按照图片安装，便于程序的编写

二、软件
软件写的较为简单，可根据自己需求改写。
大致思路是轮询+定时器，每一个动作都是都是一个循环，具体请查看代码
